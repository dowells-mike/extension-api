Tank Trouble Player Ratings API

This API provides access to player ratings data for Tank Trouble.

Endpoints

GET /api/getPlayerRatings

Get all player ratings data.

curl https://your-netlify-site.netlify.app/api/getPlayerRatings

Response Example:

{
  "players": [
    {
      "username": "Vipro",
      "tier": "a+",
      "overall": 80,
      "aiming": 83,
      "dodging": 82,
      "reflexes": 76,
      "movement": 79,
      "awareness": 76,
      "reputation": 6,
      "trait1": "aimbot",
      "trait2": "snipe",
      "trait3": "smart",
      "moderator": "",
      "ttoc": ""
    },
    // More players...
  ],
  "lastUpdated": "2023-08-02T12:34:56.789Z"
}

GET /api/getPlayerRatings?username=playername

Get ratings for a specific player by username.

curl https://your-netlify-site.netlify.app/api/getPlayerRatings?username=Vipro

Response Example:

  "player": {

Direct JSON Access

You can also directly access the JSON file at:

https://your-netlify-site.netlify.app/playerRatings.json

How to Update Data

The player ratings data is generated from your XLSM file using a build script. To update the data:

  1. Update your XLSM file with new player data
  2. Commit and push your changes to your GitHub repository
  3. Netlify will automatically rebuild and deploy the updated data
